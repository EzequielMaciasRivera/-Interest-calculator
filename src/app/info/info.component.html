Valores de entrada

Inversión inicial

Cantidad con la que se cuenta en este momento. Es decir, la cantidad con la que se empezará la inversión.
Esta cantidad debe ser mayor o igual $1,000.00

Aportación anual

¿Cuánto se puede destinar a la inversión al año?
Esta cantidad deber ser la aportación mensual destinada para la inversión multiplicada por 12.
Este valor debe ser mayor o igual a cero.
Si el valor no se proporciona para el cálculo, este deber ser considerado como cero.

% De incremento a la aportación anual

Debido a la inflación, ¿cuánto incrementará cada año la aportación?
Este valor debe ser mayor o igual a cero.
Si el valor no se proporciona para el cálculo, este deber ser considerado como cero.

Años de inversión

Duración en años de la inversión, este valor debe ser mayor a cero.

Rendimiento de inversión

Porcentaje de rendimiento de la inversión ficticia, esta debe ser mayor a cero.
Por ejemplo: Ahorradores (Cetes) 5% a 7% anual, inversionistas en Bolsa (renta variable) 10-20% anual.
